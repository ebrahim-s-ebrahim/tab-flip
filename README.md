# Tab Flip

Alt+Tab for your browser — instantly switch to your last active tab.

## How it works

Tab Flip keeps a two-element stack: **current tab** and **previous tab**. Press the shortcut and you flip between them, just like Alt+Tab on your desktop.

Works across windows. Handles closed tabs gracefully. No permissions beyond `tabs`.

## Install

1. Open `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked** and select this folder

## Shortcut

| Action | Default |
|--------|---------|
| Flip to last tab | `Alt+Q` |

Change it at `chrome://extensions/shortcuts`.
